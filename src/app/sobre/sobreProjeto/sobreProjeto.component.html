Sobre o projeto

Este projeto tem como objetivo o desenvolvimento de uma plataforma interativa que sirva como um guia prático para a criação de gráficos utilizando os princípios da Linguagem Simples. A proposta é melhorar a comunicação visual de dados e informações, tornando-as mais claras, compreensíveis e acessíveis a diversos públicos. A plataforma permitirá que os usuários explorem exemplos de gráficos bem e mal construídos, aprendendo boas práticas de design para tornar a visualização de dados mais intuitiva. Além de ser um recurso de consulta, a plataforma tem como objetivo ser uma ferramenta de aprendizado contínuo, fornecendo orientação sobre como aplicar os princípios da Linguagem Simples na criação de gráficos eficazes. A interação com a plataforma será guiada, permitindo que os usuários adaptem suas criações conforme as necessidades do público-alvo e o contexto da informação. Dessa forma, o projeto visa contribuir para uma comunicação mais eficiente e inclusiva, possibilitando que qualquer pessoa, independentemente de suas habilidades, consiga criar gráficos claros, objetivos e acessíveis.


Referênciais

As principais referencias utilizadas durante a criação desse projeto foram os livros Graficos em Linguagem Simples de Rodrigo Oliveira, Claudia Capelli e Jonice Olivera; O Livro Storytelling com Dados de Cole Nussbaumer Knaflic e os padrões de Linguagem Simples da International Plain Language Federation que versa sobre a ISO 24495-1-2023 que trata de Linguagem Simples.
