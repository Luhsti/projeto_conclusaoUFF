Aproveite “Espaços em branco” para deixar a visualização mais agradável

Esses espaços são importantes para deixar zonas de descanso para olhar do leitor, funcionam como um elemento de destaque ao ter apenas um elemento cercado de um espaço em branco, entre outras funções.

Aplicação do conceito

Lembre-se de deixar as margens livres, não deixar elementos muito próximos e aproveitar bem os espaços vazios para dar ênfase.
