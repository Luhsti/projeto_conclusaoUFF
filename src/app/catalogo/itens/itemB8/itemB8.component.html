Garanta que as cores funcionam para pessoas com deficiência ou daltonismo

No Brasil, cerca de 8% da população possui alguma dificuldade em distinguir certos tipos de cores, por exemplo, variações de azul e amarelo ou azul e verde. A mais comum, principalmente em homens, é o daltonismo que envolve a incapacidade de distinguir entre tons de vermelho e verde.

Para não prejudicar a interpretação desses leitores, faça pequenas modificações nas cores para que elas tenham contrastes seguros para todas as formas de deficiência de visão colorida.

Aplicação do conceito

Apenas escolher uma boa paleta não garante sucesso! É possível usar uma delas e mesmo assim produzir uma figura que algum daltônico não consiga ler. Logo, você pode testar diretamente com usuários ou realizar simulações para ter uma ideia de como elas podem ser para uma pessoa com alguma deficiência. Alguns aplicativos podem te ajudar com isso:

Apresenta uma série de filtros para auxiliar as pessoas a selecionar boas paletas de cores, inclusive opções seguras para daltônicos;

Gerador de paletas e combinações de cores para designers que podem ser editadas de acordo com os vários tipos de daltonismo.

Color Blindness Simulator ou Coblis é uma página web onde você pode enviar uma imagem do seu gráfico que será convertida para simular a visão daltônica.

Extensão para o navegador Firefox que adiciona um filtro a todas as páginas acessadas simulando como um daltônico visualiza as informações;

Extensão para o navegador Google Chrome para testar instantaneamente sua página da web com diferentes tipos de deficiência de visão como o daltonismo.
